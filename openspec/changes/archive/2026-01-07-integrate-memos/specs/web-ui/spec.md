## ADDED Requirements
### Requirement: Memos Input Interface
The system SHALL provide a minimalist webpage for entering memos.

#### Scenario: User saves a memo
- **WHEN** user types text into the input field AND clicks the "Save" button
- **THEN** the application SHALL send the text to the configured Memos instance AND clear the input on success.

#### Scenario: Visual Design
- **WHEN** the page is displayed
- **THEN** it SHALL use a beautiful, minimalist aesthetic (e.g., centered layout, clean typography).
